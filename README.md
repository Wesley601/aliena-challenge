## Descrição

Rest api feita em [Nest](https://github.com/nestjs/nest) framework.

## Requerimentos

```
Docker e docker-compose
node versão 16
npm versão 8
nest cli
```

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
